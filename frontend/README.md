AI Demand Forecasting System 
----------------------------


An AI-powered Demand Forecasting web application that predicts future product demand using historical sales data and provides analytics through an interactive dashboard.

Features

User Authentication (JWT Login/Register)
Dataset Upload
AI-Based Demand Forecasting
Interactive Dashboard
Sales Analytics
Forecast Accuracy Monitoring
Reports Generation
Responsive Modern UI

Tech Stack

Frontend
React.js
Tailwind CSS
Axios
Recharts
Backend
FastAPI
MySQL
SQLAlchemy
JWT Authentication
Pandas
Scikit-learn / Prophet

Run Backend
uvicorn app.main:app --reload

Backend URL:

http://127.0.0.1:8000

Run Frontend

npm run dev
